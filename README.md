# HomeDecor Furniture

HomeDecor Furniture is a professional frontend React application for browsing and managing home furniture and d√©cor items.  
Built using React.js, Tailwind CSS, DaisyUI, React Router, Context API, and LocalStorage.

## Live Demo
[HomeDecor Furniture App](https://homedecor-coms.netlify.app/)

## Features
- React Router for seamless page navigation
- Product listing and detailed product pages
- Cart functionality with LocalStorage persistence
- Global state management using Context API
- Responsive UI with Tailwind CSS & DaisyUI
- Modern and user-friendly design

## Tech Stack
- Frontend: React.js  
- UI: Tailwind CSS, DaisyUI  
- Routing: React Router  
- State Management: Context API  
- Persistence: LocalStorage  
- Deployment: Netlify
